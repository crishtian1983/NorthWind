OLTP - Procesamiento de Transacciones

Se desarrollaron todos los procedimientos almacenados necesarios para alimentar nuestro Data Warehouse desde las bases transaccionales (OLTP).

DW - Data Warehouse

Se diseñó y creó la estructura del Data Warehouse, incluyendo dimensiones, tablas de hechos y procedimientos para integrarlos. También se implementó un esquema Staging con sus respectivas tablas para gestionar datos intermedios.

ETL - Extracción, Transformación y Carga

Se implementaron paquetes ETL para cada dimensión y tabla de hechos, asegurando la correcta transformación y carga de los datos hacia el DW.

OLAP - Análisis Multidimensional

Se realizó el análisis OLAP utilizando Analysis Services, creando medidas como total de ventas, productos vendidos, descuentos aplicados, promedio de valor por pedido, entre otros.

Visualización con Power BI

La información fue visualizada en Power BI conectando con Analysis Services, permitiendo explorar métricas clave para la toma de decisiones del negocio.
